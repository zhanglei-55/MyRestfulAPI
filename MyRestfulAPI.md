## POST GetToken

POST /login

1.通过mairadb/mysql用户名和密码获取Token
2.实现身份认证

> Body 请求参数

```json
{
  "username": "微微一笑",
  "password": "123123"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Content-Type|header|string| 否 |RestfulAPI是一种返回的数据是json格式的API|
|body|body|object| 否 |none|

> 返回示例

> 200 Response

```json
{}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|成功|Inline|
